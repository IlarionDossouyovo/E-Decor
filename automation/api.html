🔌 Stack Decor API

API REST pour les agents E-Décor

GET /api/agents

http://localhost:3000/api/agents

Liste tous les agents disponibles

{"agents":[ {"id":"stack","name":"Stack Decor","status":"active"}, ... ]}

GET /api/agents/:id

http://localhost:3000/api/agents/stack

Détails d'un agent spécifique

{"id":"stack","name":"Stack Decor","role":"Principal","capabilities":[...]}

POST /api/chat

http://localhost:3000/api/chat

Envoyer un message à un agent

{"agent":"stack","message":"Bonjour","context":{}}

GET /api/analytics

http://localhost:3000/api/analytics

Statistiques en temps réel

{"sales":12450,"orders":47,"cart":265}

GET /api/health

http://localhost:3000/api/health

Vérifier le statut de l'API

{"status":"ok","ollama":"connected","timestamp":"..."}
